# PortfolioBot - Калькулятор Портфеля Активов

Этот проект представляет собой C++ приложение для расчета суммарной стоимости портфеля активов, интегрированное с внешним API для получения актуальных цен. Проект разработан с модульной архитектурой, что облегчает его расширение и поддержку.

## Возможности

- Получение актуальных цен активов с использованием внешнего API (Twelvedata).
- Расчет общей стоимости портфеля, включая различные валюты и типы активов (например, криптовалюты, фиатные деньги, золото).
- Гибкая конфигурация через JSON файлы.
- Модульная структура для легкой модификации и расширения (например, интеграции с Telegram ботом).

## Структура Проекта
```
portfolio-bot/
├── build/                 #Сюда будут складываться скомпилированные файлы (создается CMake)

│   └── portfolio.json

│   └── config.json

│   ├── api_client.cpp

│   └── main.cpp
```

## Предварительные требования

**Для сборки и запуска проекта вам понадобится:**

* Компилятор C++: Поддерживающий стандарт C++17 (например, GCC, Clang, MSVC).
* CMake: Версии 3.10 или выше.
* Библиотека cURL: Для выполнения HTTP-запросов. Установите её через системный менеджер пакетов (например, sudo apt-get install libcurl4-openssl-dev для Debian/Ubuntu, brew install curl для macOS).
* API ключ Twelvedata: Зарегистрируйтесь на Twelvedata для получения API ключа.

## Установка и Сборка

2. **Клонируйте репозиторий:**

```bash
git clone https://github.com/ВАШ_НИКНЕЙМ/portfolio-bot.git
cd portfolio-bot
```

2. **Создайте файлы конфигурации:**

* Создайте файл config/config.json:
```JSON
        {
            "twelvedata_apikey": "ВАШ_API_КЛЮЧ_TWELVEDATA",
        }
```

* Создайте файл data/portfolio.json с данными вашего портфеля. Пример:
```JSON
        {
            "RUB": "1000.00",
            "BTC/USD": "0.05",
            "ETH/USD": "0.1",
            "USD/RUB": "1.0",
            "XAU/USD": "0.01"
        }
```
3. **Установите nlohmann/json:**
    Поместите файл `json.hpp` из библиотеки [nlohmann/json](https://github.com/azadkuh/nlohmann_json_release) в директорию `vendor/nlohmann/`.

4. Сборка проекта с помощью CMake:
```Cmake
    mkdir build
    cd build
    cmake ..
    cmake --build .
```
## Использование

После успешной сборки, исполняемый файл будет находиться в директории `build/` (или `build/Debug/` на Windows). Запустите его из корневой директории проекта:
```
./build/portfolioBot
```
Программа выведет текущую стоимость вашего портфеля в консоль.
